# Zabbix Community Templates

Welcome to the **Zabbix Community Templates** repository.

This repository provides a collection of Zabbix templates developed and maintained to simplify the monitoring of various hardware, operating systems, storage solutions, and network devices.

## Purpose

The goal of this repository is to provide the Zabbix community with free, ready-to-use templates that can be easily imported into Zabbix.

Templates are designed to:

- Support Zabbix 7.x
- Follow Zabbix best practices
- Be easy to deploy
- Be maintained and improved over time

## Repository Structure


Each folder contains templates related to a specific vendor or technology.

## Compatibility

Unless otherwise specified, templates are developed for:

- Zabbix 7.4 or newer
- SNMP v2/v3
- Agent / Agent2 where applicable
